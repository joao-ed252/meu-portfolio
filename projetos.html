API Nexus

O projeto em um escopo geral consiste no desenvolvimento de um sistema voltado à análise e disponibilização de dados de exportações internacionais realizadas pelo município de São Paulo, com foco na transparência, organização e acessibilidade das informações para órgãos públicos, pesquisadores e sociedade em geral.

O meu papel de time de desenvolvimento nesse projeto foi fazer uma filtragem e listagem dos dados de exportação do município de São Paulo, listando por nome do Município, Estado, código SH4 e nome do produto e valor agregado, utilizando a linguagem Python e a biblioteca Pandas por meio do Google Colab. Com os dados filtrados torna possível fazer uma análise mais detalhada e gerar gráficos para facilitar a visualização das informações.
